# Frontend Architecture

This Vite React app keeps application code under `src/`.

- `apis/` contains data access functions. The current venue API wraps local mock data and can later call the backend.
- `components/common/` contains reusable shared components.
- `components/sections/` contains homepage sections.
- `data/` contains static seed/mock data.
- `layouts/` contains app layout wrappers.
- `pages/` contains route-level screens.
- `routes/` contains route definitions.

Archived Next.js/shadcn starter files from the generated template live in `legacy-next-scaffold/`.
